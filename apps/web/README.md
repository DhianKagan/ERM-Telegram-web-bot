# Веб‑интерфейс

Этот каталог содержит переписанный фронтенд на базе **React**, **Vite** и **Tailwind**.

## Команды

- `npm run dev` — запуск режима разработки
- `npm run build` — сборка статических файлов
- `npm run preview` — просмотр собранной версии

## UI/UX стандарты для Настроек

### ActionBar

- Располагается поверх содержимого и использует фон `var(--ui-actionbar-bg)` с тенью `var(--ui-shadow-surface)`.
- Отступы: 20 px по вертикали и 24 px по горизонтали, закругление — `var(--ui-actionbar-radius)`.
- В шапке допускается максимум две строки текста; вспомогательные элементы размещаются справа и выравниваются по базовой линии заголовка.

### Таблица

- Шапка таблицы закрепляется классом `.table-sticky` и использует фон `var(--ui-table-sticky-bg)`.
- Строки списка имеют минимальную высоту 48 px, текстовые поля выравниваются по левому краю, числовые — по правому.
- Дополнительные фильтры отображаются под ActionBar и используют компоненты chip (`.ui-chip`) для быстрой активации.

### Состояния

- Состояние загрузки отображается лоадером в ActionBar и skeleton-строками таблицы.
- Пустое состояние содержит иконку, заголовок и кнопку действия (`.ui-btn`) с явным призывом.
- Ошибки выводятся внутри ActionBar с фоном `rgba(240, 68, 56, 0.12)` и описанием шага по восстановлению.
