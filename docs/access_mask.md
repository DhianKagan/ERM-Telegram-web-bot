# Маски доступа

Этот файл — короткая памятка по числовым access mask. Каноническое описание ролей, прав, коллекции `roles` и связанных сценариев сопровождения хранится в [`docs/permissions.md`](permissions.md).

## Быстрая таблица

| Константа            | Значение | Назначение                         |
| -------------------- | -------- | ---------------------------------- |
| `ACCESS_USER`        | `1`      | Обычный пользователь               |
| `ACCESS_ADMIN`       | `2`      | Администратор                      |
| `ACCESS_MANAGER`     | `4`      | Менеджер или промежуточная роль    |
| `ACCESS_TASK_DELETE` | `8`      | Расширенный уровень удаления задач |

## Что важно помнить

- Роль `admin` сочетает `ACCESS_ADMIN | ACCESS_MANAGER` и имеет маску `6`.
- Удаление задач требует отдельной маски `ACCESS_TASK_DELETE = 8`.
- Полная operational-документация по RBAC и роли `roles` поддерживается в [`docs/permissions.md`](permissions.md).
- Исходный код масок и проверок находится в `apps/api/src/utils/accessMask.ts` и `apps/api/src/auth/roles.guard.ts`.
