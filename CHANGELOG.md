# История изменений

- Восстановлены workflow GitHub Actions: CodeQL локализован, деплой и релиз проверяют секреты Railway и собирают проект через pnpm.

- Антивирус для вложений переключён на clamd через `clamdjs`, добавлены настройки окружения и unit-тесты.
