# Архив документации

В этом каталоге собраны устаревшие и исторические документы, сохранённые для справки.

## Что попадает в архив

- старые гайды, которые больше не отражают текущую архитектуру;
- разовые черновики и исследовательские заметки;
- дублирующие документы, конфликтующие с каноническими источниками.

## Пример

Файл `codex_system_prompt.md` перенесён в архив, чтобы не дублировать инструкции из `AGENTS.md` и `.openai/assistant_instructions.json`.
