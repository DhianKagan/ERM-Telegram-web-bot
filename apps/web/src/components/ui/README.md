# UI компоненты

- Кнопки: `apps/web/src/components/ui/button.tsx`.
- Поля ввода и выпадающие списки: `input.tsx` и `select.tsx`.
- Группы полей и карточки: `form-group.tsx` и `card.tsx`.
- Таблицы: для примитивов используйте `table.tsx`, для продвинутых таблиц с пагинацией и виртуализацией — `apps/web/src/components/DataTable.tsx`.
