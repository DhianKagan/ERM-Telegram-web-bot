# Шаг B1 — консолидация внутренних пакетов в `packages/*`

Дата: 12 Feb 2026 (Europe/Kyiv)
Источник: `docs/monorepo-unification-task-breakdown.md`.

## Что сделано

- Подтверждено, что в `apps/*` остаются только запускаемые сервисы: `api`, `web`, `worker`.
- Создан новый внутренний переиспользуемый пакет `packages/utils` (`@erm/utils`).
- В `@erm/utils` вынесены общие гео-утилиты (обёртки над `shared` и единая нормализация env-настроек `ROUTE_*`).
- В `apps/api` и `apps/worker` локальные `src/utils/geo.ts` переведены на реэкспорт из `@erm/utils`.

## Результат

- Повторяющийся код утилит между приложениями сокращён.
- Переиспользуемая логика перемещается из `apps/*` в `packages/*` без изменения внешнего API приложений.
- Подготовлена база для последующих шагов B2/B3 (границы пакетов и оркестрация).
